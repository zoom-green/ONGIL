# ON:길 사용 가이드

ON:길은 강릉에서 밤길을 이동할 때 더 안전한 경로를 찾고, 위급 상황에 빠르게 대응할 수 있도록 돕는 모바일 웹앱입니다.

이 가이드는 서비스를 처음 이용하는 사람이 모든 기능을 빠짐없이 사용해볼 수 있도록 작성되었습니다.

## 1. 서비스에서 할 수 있는 일

ON:길에서 경험할 수 있는 핵심 기능은 다음과 같습니다.

- 현재 위치 기반 지도 보기
- 출발지와 도착지 검색
- 지도에서 직접 출발지/도착지 지정
- 빠른길과 안심길 비교
- CCTV, 음식점/카페, 편의점, 경찰서/지구대, 소방서, 안전지킴이집, 응급의료시설, 비상벨 확인
- 선택한 경로 주변의 안전시설 확인
- 이동 시작 후 경로 이탈 여부 확인
- 보호자 연락처 저장
- 이동 중 보호자에게 위치 공유 문자 보내기
- SOS 긴급 신고 화면 사용
- 112 전화 연결
- 보호자 긴급 문자 보내기
- AI 음성 동행 통화 사용
- 엄마, 아빠, 오빠 페르소나 선택

## 2. 처음 시작하기

1. ON:길 URL에 접속합니다.
2. 위치 권한 요청이 뜨면 허용합니다.
3. 지도가 열리면 현재 위치가 파란 점으로 표시됩니다.
4. 위치 권한을 허용하지 않으면 기본 위치인 강릉 중심 좌표를 기준으로 지도가 열릴 수 있습니다.

위치 권한은 현재 위치 기반 출발지 설정, 이동 추적, 보호자 위치 공유, SOS 위치 전달에 필요합니다.

## 3. 화면 구성

처음 화면에는 다음 영역이 있습니다.

- 상단: 서비스 이름, 설정 버튼
- 검색 영역: 출발지 입력칸, 도착지 입력칸
- 안전시설 토글: CCTV, 카페/음식점, 편의점, 경찰서/지구대, 소방서, 안전지킴이집, 응급의료시설, 비상벨
- 지도: 현재 위치, 선택 위치, 안전시설, 경로 표시
- 왼쪽 아래: SOS 버튼
- 오른쪽 아래: AI 음성 대화 버튼
- 하단: 경로 검색 후 빠른길/안심길 카드

## 4. 출발지 설정하기

출발지는 기본적으로 현재 GPS 위치로 설정됩니다.

다른 곳에서 출발하고 싶다면:

1. 출발 입력칸을 누릅니다.
2. 장소명이나 주소를 입력합니다.
3. 검색 결과에서 원하는 장소를 선택합니다.

다시 현재 위치로 되돌리고 싶다면:

1. 출발 입력칸 옆의 `GPS` 버튼을 누릅니다.
2. 출발지가 현재 위치로 초기화됩니다.

지도에서 직접 출발지를 지정할 수도 있습니다.

1. 지도에서 원하는 위치를 누릅니다.
2. 아래에 선택한 위치 패널이 열립니다.
3. `출발지로 설정`을 누릅니다.

## 5. 도착지 설정하기

1. 도착 입력칸을 누릅니다.
2. 장소명이나 주소를 입력합니다.
3. 검색 결과에서 목적지를 선택합니다.
4. 목적지를 선택하면 자동으로 경로 분석이 시작됩니다.

지도에서 직접 도착지를 지정할 수도 있습니다.

1. 지도에서 원하는 위치를 누릅니다.
2. 아래에 선택한 위치 패널이 열립니다.
3. `도착지로 설정`을 누릅니다.

## 6. 장소 검색 이해하기

ON:길의 검색은 한 가지 데이터만 쓰지 않습니다.

검색 결과는 다음 정보를 함께 활용합니다.

- 현재 위치 주변 Kakao 장소 검색
- Kakao 전체 장소 검색
- Kakao 주소 검색
- Tmap 장소 검색

검색 결과에 거리가 표시되면 현재 위치에서 가까운 장소를 우선 확인할 수 있습니다.

## 7. 빠른길과 안심길 보기

도착지를 선택하면 하단에 경로 카드가 나타납니다.

### 빠른길

빠른길은 이동 시간과 거리를 우선으로 계산한 경로입니다.

확인할 수 있는 정보:

- 예상 이동 시간
- 총 거리
- Tmap 기준 보행자 경로

### 안심길

안심길은 안전시설 주변을 더 많이 지나도록 계산한 경로입니다.

안심길은 다음 요소를 고려합니다.

- CCTV
- 가로등/보안등
- 편의점
- 카페/음식점
- 경찰서/지구대
- 소방서
- 안전지킴이집
- 응급의료시설
- 비상벨

안심길 카드에서는 경로 주변의 안전시설 개수도 함께 확인할 수 있습니다.

## 8. 안심길이 계산되는 방식

ON:길은 먼저 가장 빠른 보행 경로를 찾습니다.

그 다음 경로 주변의 안전시설을 분석하고, 안전시설 가까이를 지나도록 여러 후보 경로를 다시 계산합니다.

최종 안심길은 다음 기준으로 선택됩니다.

1. CCTV와 가로등이 부족한 어두운 구간이 적은 경로
2. 편의점, 경찰서, 소방서, 안전지킴이집 같은 안전거점이 많은 경로
3. 위 조건이 비슷하면 이동 거리가 짧은 경로

즉, 안심길은 단순히 가장 짧은 길이 아니라 밤길에서 더 안심하고 걸을 수 있도록 설계된 경로입니다.

## 9. 안전시설 지도에서 확인하기

지도 위 안전시설 토글을 켜면 해당 시설이 지도에 표시됩니다.

사용 방법:

1. 지도 위쪽의 안전시설 버튼을 확인합니다.
2. 보고 싶은 안전시설을 누릅니다.
3. 버튼이 활성화되면 지도에 해당 시설 마커가 표시됩니다.
4. 다시 누르면 해당 시설 마커가 숨겨집니다.

각 안전시설의 역할은 다음과 같습니다.

### CCTV

주변에 감시 시설이 있는지 확인할 수 있습니다.

### 카페/음식점

밤길 이동 중 사람이 있거나 도움을 요청할 수 있는 장소를 확인할 수 있습니다.

### 편의점

늦은 시간에도 열려 있을 가능성이 높은 안전거점을 확인할 수 있습니다.

### 경찰서/지구대

위급 상황에서 직접 도움을 요청할 수 있는 치안 거점입니다.

### 소방서

응급 상황이나 화재 등 긴급 상황에서 도움을 받을 수 있는 공공 안전거점입니다.

### 안전지킴이집

아동과 보행자가 위급할 때 도움을 요청할 수 있는 지역 안전거점입니다.

### 응급의료시설

부상이나 건강 문제가 생겼을 때 이용할 수 있는 의료 거점입니다.

### 비상벨

위험 상황에서 주변 도움을 요청할 수 있는 시설입니다.

## 10. 경로 주변 안전시설 확인하기

안심길을 선택하면 경로 가까이에 있는 안전시설이 함께 표시됩니다.

일부 경로 증거 마커는 지도를 확대했을 때 더 잘 보입니다.

사용 방법:

1. 도착지를 선택해 경로를 계산합니다.
2. `안심길` 카드를 선택합니다.
3. 지도를 확대합니다.
4. 경로 주변에 어떤 안전시설이 있는지 확인합니다.

## 11. 이동 시작하기

경로를 선택한 뒤 실제 이동을 시작하려면:

1. 하단 경로 패널에서 원하는 경로를 선택합니다.
2. `이동 시작` 버튼을 누릅니다.
3. 버튼이 `이동 추적 중`으로 바뀝니다.
4. 현재 위치가 경로를 따라 이동하는지 확인합니다.

## 12. 경로 이탈 확인하기

이동 시작 후 ON:길은 현재 위치와 선택한 경로 사이의 거리를 비교합니다.

상태는 두 가지로 표시됩니다.

- `경로 따라 이동`: 현재 위치가 선택한 경로 근처에 있음
- `경로 이탈`: 현재 위치가 선택한 경로에서 많이 벗어남

경로 이탈 상태는 보호자 위치 공유 문자에도 함께 포함될 수 있습니다.

## 13. 보호자 연락처 설정하기

보호자에게 위치를 공유하거나 긴급 문자를 보내려면 연락처를 먼저 저장합니다.

사용 방법:

1. 상단의 `설정` 버튼을 누릅니다.
2. 보호자 연락처 1, 보호자 연락처 2를 입력합니다.
3. 위치 공유 알림 주기를 선택합니다.
4. `저장하기`를 누릅니다.

보호자 연락처는 최대 2명까지 등록할 수 있습니다.

## 14. 위치 공유 알림 주기 설정하기

설정 화면에서 위치 공유 알림 주기를 선택할 수 있습니다.

선택 가능한 주기:

- 2분
- 4분
- 8분

이동 시작 후 설정한 시간이 지나면 보호자에게 위치 공유 문자를 보낼 수 있는 알림이 뜹니다.

## 15. 보호자에게 위치 공유 문자 보내기

이동 중 위치 공유 시간이 되면 화면 위쪽에 알림이 뜹니다.

사용 방법:

1. `보호자에게 위치 공유 시간이 됐습니다` 알림을 확인합니다.
2. `문자 보내기` 버튼을 누릅니다.
3. 보호자에게 현재 위치와 이동 상태가 포함된 문자를 보냅니다.

문자에는 다음 내용이 포함됩니다.

- 이동 중인 경로 종류
- 목적지
- 현재 위치 Google Maps 링크
- 경로를 따라 이동 중인지, 경로를 이탈했는지

웹 브라우저에서는 문자 앱 작성 화면이 열릴 수 있고, Android 앱에서는 문자 전송 기능을 사용할 수 있습니다.

## 16. SOS 긴급 신고 사용하기

위급 상황에서는 지도 왼쪽 아래의 `SOS` 버튼을 누릅니다.

사용 방법:

1. `SOS` 버튼을 누릅니다.
2. 긴급 신고 화면이 열립니다.
3. 5초 카운트다운이 시작됩니다.
4. 취소하지 않으면 112 전화 연결 화면으로 이동합니다.

카운트다운 중 실수로 눌렀다면 취소할 수 있습니다.

## 17. 112 전화 연결하기

SOS 화면에서 카운트다운이 끝나면 `tel:112` 전화 연결이 실행됩니다.

휴대폰에서는 전화 앱이 열리고, 사용자가 통화를 진행할 수 있습니다.

전화 연결 후에도 화면에는 112 신고 카드와 보호자 SMS 카드가 표시됩니다.

## 18. 보호자 긴급 문자 보내기

SOS 화면에서 보호자에게 긴급 문자를 보낼 수 있습니다.

사용 방법:

1. SOS 화면에서 보호자 SMS 카드를 확인합니다.
2. `문자 보내기` 버튼을 누릅니다.
3. 현재 위치 링크가 포함된 긴급 문자를 보호자에게 보냅니다.

보호자 연락처가 설정되어 있지 않으면 먼저 설정에서 연락처를 등록해야 합니다.

## 19. AI 음성 동행 사용하기

혼자 걷는 동안 대화 상대가 필요하면 AI 음성 동행을 사용할 수 있습니다.

사용 방법:

1. 지도 오른쪽 아래의 `AI 음성 대화` 버튼을 누릅니다.
2. 통화할 페르소나를 선택합니다.
3. `통화하기` 버튼을 누릅니다.
4. 마이크 권한 요청이 뜨면 허용합니다.
5. AI와 음성으로 대화합니다.

AI 음성 동행은 실제 전화처럼 듣고 말하는 방식으로 동작합니다.

## 20. AI 페르소나 선택하기

AI 음성 동행에는 세 가지 페르소나가 있습니다.

### 엄마

따뜻하고 걱정 많은 말투로 대화합니다.

### 아빠

무뚝뚝하지만 든든한 말투로 대화합니다.

### 오빠

장난스럽지만 챙겨주는 말투로 대화합니다.

상황이나 기분에 따라 원하는 동행자를 선택하면 됩니다.

## 21. AI 통화 화면 사용하기

AI 통화 화면에서는 다음 상태를 볼 수 있습니다.

- 연결 중
- 듣는 중
- 말하는 중
- 통화 시간

통화 중 화면을 작게 줄이고 싶으면 축소 버튼을 누릅니다.

통화를 끝내려면 `통화 종료` 버튼을 누릅니다.

## 22. 지도 클릭 기능 사용하기

지도에서 특정 위치를 누르면 아래쪽에 위치 선택 패널이 열립니다.

이 패널에서 할 수 있는 일:

- 선택한 위치를 출발지로 설정
- 선택한 위치를 도착지로 설정
- 선택을 취소

주소를 정확히 모르는 장소도 지도에서 직접 눌러 사용할 수 있습니다.

## 23. 경로 패널 접기와 다시 보기

경로를 계산한 뒤 지도 화면을 넓게 보고 싶다면 경로 패널을 접을 수 있습니다.

사용 방법:

1. 경로 패널의 `접기` 버튼을 누릅니다.
2. 패널이 접히고 지도 영역이 넓어집니다.
3. 다시 보고 싶으면 `경로 보기` 버튼을 누릅니다.

## 24. 기능별 사용자 역할

### 일반 보행자

주요 사용 기능:

- 목적지 검색
- 안심길 선택
- 안전시설 확인
- 이동 시작
- 경로 이탈 확인
- AI 음성 동행
- SOS 신고

효과:

- 어두운 길을 피하고 더 안심되는 경로를 선택할 수 있습니다.
- 혼자 걷는 불안감을 줄일 수 있습니다.
- 위급 상황에서 빠르게 대응할 수 있습니다.

### 보호자

직접 앱을 조작하지 않아도 문자로 사용자의 상태를 확인할 수 있습니다.

받을 수 있는 정보:

- 현재 위치 링크
- 목적지
- 이동 상태
- 긴급 상황 알림

효과:

- 사용자가 어디쯤 있는지 확인할 수 있습니다.
- 경로 이탈이나 긴급 상황을 빠르게 파악할 수 있습니다.

### 심사위원 또는 체험자

추천 체험 순서:

1. 위치 권한 허용
2. 도착지 검색
3. 안심길과 빠른길 비교
4. 안전시설 토글을 하나씩 켜보기
5. 지도에서 직접 도착지 설정
6. 이동 시작 버튼 누르기
7. 보호자 연락처 설정 화면 확인
8. SOS 버튼 눌러 긴급 화면 확인
9. AI 음성 대화 실행

이 순서대로 체험하면 ON:길의 모든 핵심 기능을 확인할 수 있습니다.

## 25. 체험 추천 시나리오

처음 이용자는 다음 흐름으로 체험하는 것을 추천합니다.

1. ON:길에 접속합니다.
2. 위치 권한을 허용합니다.
3. 도착지에 강릉 시내 장소를 검색합니다.
4. 안심길과 빠른길을 비교합니다.
5. CCTV, 편의점, 경찰서, 소방서, 안전지킴이집 토글을 켭니다.
6. 경로 주변에 어떤 안전시설이 있는지 확인합니다.
7. `이동 시작`을 누릅니다.
8. 설정에서 보호자 연락처와 위치 공유 주기를 확인합니다.
9. `AI 음성 대화`를 눌러 원하는 페르소나와 통화합니다.
10. 지도 왼쪽 아래의 `SOS` 버튼을 눌러 긴급 신고 화면을 확인합니다.
11. 실수로 누른 상황이라면 카운트다운 중 취소합니다.

## 26. 사용 시 알아둘 점

- 위치 권한을 허용해야 현재 위치 기반 기능을 제대로 사용할 수 있습니다.
- 문자 전송은 기기와 실행 환경에 따라 동작 방식이 다를 수 있습니다.
- 웹에서는 문자 앱 작성 화면이 열리고, 실제 전송은 사용자가 확인해야 할 수 있습니다.
- AI 음성 동행은 마이크 권한이 필요합니다.
- 안전시설 데이터는 공공데이터와 지도 API를 기반으로 하므로 실제 현장 상황과 다를 수 있습니다.
- 위급 상황에서는 앱 안내보다 실제 112 신고와 주변 도움 요청이 우선입니다.

## 27. 한 줄 요약

ON:길은 목적지까지 단순히 빠르게 가는 길이 아니라, 주변 안전시설과 위급 대응 수단을 함께 확인하며 더 안심하고 이동할 수 있도록 돕는 야간 보행 안전 서비스입니다.
