- 데모 URL
- GitHub 저장소 링크
- 발표 자료 PDF

## 제출 전 확인

- 데모 URL이 실제로 열리는지 확인
- 지도 로딩 확인
- 장소 검색 확인
- 빠른길/안심길 표시 확인
- 안전시설 토글 확인
- SOS 버튼 확인
- AI 음성 대화 진입 확인
- `.env`가 GitHub에 올라가지 않았는지 확인
- README의 실행 방법과 환경변수 안내 확인
